# SiciliaRicambi SEO Tool (con inserimento API da browser)

## Come si usa

1. Avvia il tool:
```
streamlit run seo_generator.py
```

2. Inserisci manualmente la tua API Key nella casella

3. Carica `Products.csv` e `Funzioni.csv`

4. Scarica i file ottimizzati per ETS SEO

## Nessun file .env necessario
